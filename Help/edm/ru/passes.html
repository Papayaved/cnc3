Задание параметров реза

  • Смещение проволоки

    Параметр изменяет контур реза для компенсации ширины проволоки.

    • Левое смещение

      При резе используется левое смещение проволоки относительно контура по направлению реза.

    • Правое смещение

      При резе используется правое смещение проволоки относительно контура по направлению реза.

  • Число проходов

    Число проходов по контуру реза, [1, 7].

  • Смещение при проходе 1..7

    Смещение проволоки относительно контура реза на 1..7 проходе.

  • Режим при проходе 1..7

    Режим работы станка на 1..7 проходе.

  • Режим линии захода

    Режим работы станка на линии захода.

  • Недорез

    • Однопроходный

      Сделать однопроходный дорез с заданным смещением и режимом реза.

    • Многопроходный

      Сделать многопроходный дорез. Смещение и режимы работы используются, как у основного реза.

    • Ширина недореза

      Длина на которую не дорезается контур детали. Параметр используется для того, чтобы деталь не выпадала при многопроходном резе замкнутого контура.

    • Использовать последний сегмент

      Использовать в качестве недореза последний сегмент контура.

    • Смещение при дорезе

      Смещение проволоки при однопроходном дорезе контура.

    • Режим при дорезе

      Режим работы при однопроходном дорезе контура.

    • Сделать паузу перед недорезом

      Сделать паузу для того, чтобы закрепить деталь. Убедитесь, что на заготовку не подано высокое напряжение!

  • Перерез

    Задание длины перереза в миллиметрах. Перерез используется для того, чтобы обработать точку захода. Режим работы станка при перерезе выбирается, как при последнем проходе.

  • Ограничение скорости реза

    Параметр задает максимальную скорость реза.
