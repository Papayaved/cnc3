Pult

Common control

Buttons

  • Home - go to the Home panel

    The button is disabled during running.

  • Break

    Wire break control turn On/Off.

  • Pump

    Cutting fluid turn On/Off.

  • Roll

    Wire roll turn On/Off.

  • High Voltage - workpiece high voltage turn On/Off

    130V or 90V, depend on low High voltage level.

  • Low High Volt. - low High Voltage level turn On/Off

    Turn on - 90 Ð’, Turn off - 130 V.

  • Cancel

    Cancel moving.

  • Hold

    Motors' holding turn On/Off.

Controls

  • Roll - wire roll velocity

    Values from 1 to 7.

  • Current - current limit

    Values from 0 to 11.

  • Width - high voltage pulse width

    Values from 2 to 99.

  • Ratio - pulses' duty cycle

    Values from 2 to 32.

  • Speed - maximum cutting speed

    Values from 0,01 to 18 mm/min.

Task panel

Buttons

  • Forward

    Move on distance dX, dY, dU, dV with given speed. Same as the G-code command
    G91 G1 X{dX} Y{dY} U{dU} V{dV} F{speed}

  • Reverse

    Move on distance dX, dY, dU, dV with given speed in reverse direction. Same as the G-code command
    G91 G1 X{-dX} Y{-dY} U{-dU} V{-dV} F{speed}

  • Set

    Set CNC position counters to X, Y, U, V values. Same as the G-code command
    G92 X{X} Y{Y} U{U} V{V}

  • Copy

    Copy current coordinates. Together with the "Back" button, it is used to return to the zero position.

  • Remain

    Calculate the difference between the given distance and current position.

  • Clear

    Clear fields dX, dY, dU, dV, X, Y, U, V.

  • Reset

    Set default parameters.

Parameters

  • Scale

    The precision of stepper motors and linear encoders is set here.

    The values are not written to the CNC. They are only used in the Pult panel.

State panel

The State panel displays:

  • Current coordinates X, Y, U, V in millimeters;;
  • Elapsed time from the start in HH:MM:SS format;
  • Estimated remaining time in HH:MM:SS format;
  • Flags:
    • PWR - no power supply 220V;
    • WB - wire break signal;
    • TO - short circuit timeout;
    • FB - feedback limit switch;
    • ALM - an alarm signal from one of the limit switches;
    • REV - the limit switch of the backward motion of the wire roll is active;
    • FWD - the limit switch of the forward motion of the wire roll is active.
