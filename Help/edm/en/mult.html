Contour multiplication

The form allows you to multiplicate a part and cut out a batch of parts in one G-code program. Parts are arranged on the workpiece in rows along the trajectory with a snake. The part must have a closed contour and 4 junction vertices, in the positive and negative directions of the X and Y axes. These vertices will be used as junction points to connect the parts to each other. A vertex is a point where two adjacent segments are connected.

  • Row direction

    Choice the main axis of parts' multiplication. The direction along the axis depends on the sign of the width between the parts.

    • X

      Rows run along the X-axis.

    • Y

      Rows run along the Y-axis.

  • Parts per row

    The number of parts in each row.

  • Rows

    The number of rows.

  • Gap between parts in each row

    Signed value. The sign determines the direction of the parts in a row.

  • Gap between rows

    Signed value. The sign determines the direction of increasing the number of rows.

  • Auto choice in the contour 4 junction verteces

    Automatically choice in the closed contour 4 existing vertices as junction points, or manually:

    • -X junction point is at the beginning of the segment with the given number

      Set the contour junction point for the negative direction of the X-axis.

    • +X junction point is at the beginning of the segment with the given number

      Set the contour junction point for the positive direction of the X-axis.

    • +Y junction point is at the beginning of the segment with the given number

      Set the contour junction point for the positive direction of the Y-axis.

    • -Y junction point is at the beginning of the segment with the given number

      Set the contour junction point for the negative direction of the Y-axis.

  • Entry Line Length

    The length of the Entry Line for the new complex contour.

  • Exit Line Length

    The length of the Exit Line for the new complex contour.

Buttons

  • OK

    Multiplicate the part and return to the Contour editor.

  • Cancel

    Cancel the action and return to the Contour editor.

  • Default

    Set default parameters.
