Application settings

  • Language

    Choosing the language of the program interface. The changes will take effect after a reboot.

CNC settings

  • Input signals levels

    In the drop-down menu, you can select presets CNC input levels for EDM (Metal) or stone cutting (Stone), or program debugging (Debug) CNC mode.

  • Calculation step

    Calculation step of the next position.

  • Use servomotors

    Generate step-dir signals instead phase signals.

  • Reverse motors X, Y, U, V

    Reverse direction of X, Y, U, V axes.

  • Swap motors X, Y and U, V

    Swap X,Y and U,V axes. The X-axis must be in the same plane as the U-axis and have the same direction. The same rule is used for the Y and V axes.

  • Use linear encoders for X, Y axes

    If linear encoders are installed, they can be used when cutting.

    • Reverse Encoder X, Y

      Reverse encoder directions. The encoder direction must match the corresponding axis.

  • Feedback enable

    Enable feedback mode. CNC keeps cutting voltage in the range between Low and High Thresholds. CNC have two voltage ranges for High Voltage and Low High Voltage.

    • High, Low thresholds

    • Rollback timeout

      The time in the short-circuit state after which the CNC rolls back to the rollback distance.

    • Rollback distance

    • Rollback speed

      Rollback speed after a short circuit.

    • Rollback attempts

      Number of attempts to start cutting after a short-circuit.

  • Acceleration

    Maximum acceleration. It's only used to cutting speed control in feedback enabled mode.

  • Deceleration

    Maximum deceleration.

  • Motors' precision

    A number of motor steps per mm for each axis.

  • Encoders' precision

    A number of encoder steps per mm for each axis.

Buttons

  • Home - go to the Home panel

  • Read - read CNC settings

  • Write - write CNC settings

  • Default - set default settings
