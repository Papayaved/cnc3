Cutting settings

Bottom panel

Buttons

  • Back

    Return to the Contour editor.

  • Open

    Open cutting settings from file.

  • Save

    Save cutting setting to the current file.

  • Save as

    Save cutting settings as a new file.

  • Default

    Set default values.

  • Generate

    Generate the G-code program based on the contour and cutting settings. After successful generation, the program goes to the Work Panel.

Cutting settings panel

  • Wire offset

    Wire width compensation. This parameter changes the contour to compensate for the width of the cutting wire.

    • Left offset

      Use wire offset to the left of the cutting direction.

    • Right offset

      Use wire offset to the right of the cutting direction.

  • Cutting times

    Number of passes along the cutting contour. Values from 1 to 7.

  • Offsets for passes 1..7

    Wire offsets for passes 1 to 7.

  • Modes for passes 1..7

    Operation modes for passes 1 to 7.

  • Entry line mode

    Entry line operation mode.

  • Workpiece Tab

    The CNC machine can stop before it reaches the end of the contour, leaving a tab (or bridge) to hold a part in the workpiece. It prevents a part from falling down. After clamping the part, you can continue cutting by pressing the "Start" button.

    • Onepass

      Cut off the tab with onepass.

    • Multi-pass

      Cut off the tab using the multi-pass mode as in the main cut.

    • Tab Width

    • Use the last segment

      Use the last contour segment as a tab.

    • Tab Offset

      Wire offset when cutting the tab with onepass.

    • Tab Mode

      Operation mode when cutting the tab with onepass.

    • Tab Pause

      Pause to clamp the part. Make sure, there is no High Voltage on the workpiece!

  • Overlap

    This parameter sets the length of the contour overcut. It's used to process the entry point.

  • Cutting Speed Limit

    Set maximum cutting speed.

  • Tapered Cutting Parameters

    Parameters for two contours cutting.

    • Distance between rollers (L)

      Distance between axes of rollers.

    • Workpiece bottom height (H)

      Distance between the axis of the lower roller and the bottom of the workpiece.

    • Workpiece thickness (T)

      The workpiece must have a rectangular profile.

    • Use the roller diameter

      Take into account the roller's diameter for tapered cutting mode.

      • Roller diameter

      • Wire diameter

      • Roller plane

        • XZ

        • YZ
