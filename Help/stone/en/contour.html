Contour Editor

Bottom Panel

Buttons

  • Home

    Go to the Home Panel.

  • Open

    Open a project.

  • Save

    Save current project.

  • Save as

    Save as a copy or new file.

  • Load DXF

    Load a cutting contour in the DXF format. Only line and arc segments are supported in the DXF file. If the DXF file contains several layers, you will be prompted to select the required layer.

  • Multiply

    Multiplicate the selected closed contour. All other contours will be removed.

  • Generate

    Set up cutting parameters and modes, and generate G-code

Contour Edit Panel

Buttons

  • New

    Create new empty contour.

  • New Entry

    Add a new cutline before the first segment. The Entry line will be attached to the first contour point marked with a red square on the plot.

  • Delete

    Delete the selected contour from the project.

  • Clear

    delete all contours from the project.

  • First

    Set selected segment or contour as the first in the list.

  • Up

    Move selected segment or contour up in the list.

  • Down

    Move selected segment or contour down in the list.

  • Last

    Set selected segment or contour as the last in the list.

  • Sort

    Automatically sort the segments in the current contour.

  • Properties

    Selected segment or contour properties. They allow you to move a segment to another contour or change the contour type.

  • Undo

    Undo the last change.

  • Меню Edit

    • Use as Entry line

      Use the selected contour segment as the Entry line of a single contour. It works only with a single contour in the project.

    • Change Direction

      Reverse the segments' order in the current contour. This operation changes the cutting direction.

    • Rotate

      Rotate all contours in the project by a given angle.

    • Flip Left-Right

      Flip all contours in the project from left to right.

    • Flip Upside-Down

      Flip all contours in the project upside down.

    • Resize

      Resize all contours in the project.
