Slicing Panel

Parameters

  • Wire Diameter

  • Workpiece Width or Diameter

  • Spacing

    The gap between the wire and the workpiece when the CNC moves to a new cut line. On the plot, this looks like an additional width of the workpiece.

  • Step

    The thickness of the cut parts. The signed value depends on the slicing direction.

  • Slices Number

  • Workpiece Profile

    • Rectangle profile

    • Round profile

  • Speed or Average Speed

    Cutting speed. For a round profile, the speed will change depending on the width of the workpiece at the current position.

  • Sections' number in the speed profile

    The number of speed sections for a workpiece with a round profile.

  • Pause

    Pause time after each slice. This parameter is used when the wire bends due to excessive cutting speed.

  • Axis

    Axis of the slicing direction. The direction also depends on the sign of the slicing step.

    • X

    • Y

Buttons

  • Reset

    Reset to default.

  • Undo

    Return last used.

  • Generate

    Generate G-code.
