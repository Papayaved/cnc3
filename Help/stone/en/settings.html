Application settings

  • Language

    Choosing the language of the program interface. The changes will take effect after a reboot.

CNC settings

  • Input signals levels

    In the drop-down menu, you can select presets CNC input levels for EDM (Metal) or stone cutting (Stone), or program debugging (Debug) CNC mode.

  • Calculation step

    Calculation step of the next position.

  • Reverse motors X, Y

    Reverse direction of X, Y axes.

  • Swap motors X, Y

    Swap X,Y axes.

  • Acceleration enable

    Enable smooth speed up at the beginning of each segment or after stopping the cut, and smooth slow down at the end of each segment.

    • Acceleration

      Acceleration value in um/sec2.

    • Deceleration

      Deceleration value in um/sec2.

Buttons

  • Home - go to the Home panel

  • Read - read CNC settings

  • Write - write CNC settings

  • Default - set default settings
