Bottom Panel

Buttons

  • Contour - Contour Editor

    Load and adjust DXF format contour, set multi-pass cut modes.

  • G-code - G-code Editor

    G-code CNC program editor.

  • Run - Work Panel

  • Settings

    Set up the program interface and CNC parameters.

  • Pult

    CNC manual control.

  • Recovery

    Recovering the last G-code program that was ended due to cancellation or a power failure.

  • Minimize

    Minimize the program and switch to the operation system desktop.

  • Shutdown

    Quit the program and turn off the computer.

List of extensions

  • Welcome Panel

    The panel displays the welcome screen of the CNC app. This screen contains information about the application and firmware versions. The CNC has two firmware for the FPGA microchip and for the microcontroller.

  • Slicing

    Generation of the toolpath for a workpiece slicing.

  • Slots

    Generation of the toolpath for cutting slots in a workpiece. This toolpath can also be used for slicing.
